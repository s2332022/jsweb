# シンプル計算ページ (フロントエンドのみ)

これは極めてシンプルなフロントエンド-onlyのサンプルページです。値や文字列を入力して、簡単な計算や文字列操作の結果を表示します。

使い方:

1. このフォルダで `index.html` をダブルクリックするか、ブラウザで開いてください。
2. テキスト入力に値を入力します。
   - 数値モード時はカンマ区切りで数を入力します（例: `1,2,3.5`）。
   - 文字列モード時はそのまま文字列を入力します（例: `こんにちは 世界`）。
3. モード（数値 / 文字列）と操作を選び、`実行` をクリックすると結果が表示されます。

含まれるファイル:

- `index.html` — 最小限の UI
- `script.js` — 前端ロジック（数値・文字列操作）
- `style.css` — とてもシンプルなスタイル

注意:

- 完全にクライアントサイドのみです。サーバーは不要です。
- 実際のユースケースではもっと入力検証が必要です。これは学習/デモ用途の最小限実装です。
